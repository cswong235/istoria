## About Istoria

A web application that serves as a gallery for my personal collection of old artifacts, trinkets and mostly rare books. Each of these entries feature a small backstory of what it is, what events it has lived through, and how did I manage to get my hands on them.

## Frameworks and Tools Used
- Laravel 12
- Inertia.js
- MySQL
- JavaScript
- Tailwind CSS + DaisyUI
- Vite
